## Core Philosophy: Veracity & Provenance

This project adheres to one strict rule: **religious text is never generated, inferred, or filled in by AI.** All datasets are gathered directly from established, authoritative public references, validated for structural integrity, and indexed with full provenance metadata including:
*   The exact source URL and direct download link.
*   Retrieved timestamp.
*   Specific content license/redistribution terms.
*   File byte size.

This guarantees absolute authenticity and reproducibility, critical for religious applications.
